# パワーワード詳細辞典

SKILL.md Section 2 の詳細版。フック・CTA・LP で使う感情を動かす言葉を6カテゴリに分類。

---

## 1. 緊急性

「今」行動する理由を作る言葉。

### 基本ワード

| ワード | 強度 | 用途 |
|--------|------|------|
| 今すぐ | 強 | CTA ボタン |
| リアルタイムで | 中 | 機能説明 |
| 見逃さない | 中 | LP ヘッドライン |
| 逃さない | 中 | LP サブコピー |
| タイミングを逃さず | 中 | 機能説明 |
| 自動で検知 | 中 | 機能説明 |

### TrendBird 実例

- 「バズるタイミングを逃しません」— LP ヒーロー
- 「リアルタイムでトレンドの盛り上がりを検知」— LP サブコピー
- 「盛り上がりをリアルタイムでキャッチ」— チュートリアル

### 日本市場での注意

- 「今すぐ〜しないと損」系の過度な煽りは逆効果
- 「自然に気づける」「見逃さずに済む」のように受動的メリットとして表現すると日本市場で刺さる
- LP では使いつつ、UI 内では控えめに

---

## 2. 安心感

不安を取り除き、行動のハードルを下げる言葉。

### 基本ワード

| ワード | 強度 | 用途 |
|--------|------|------|
| 無料 | 強 | CTA・LP |
| かんたん | 強 | LP・チュートリアル |
| クレジットカード不要 | 強 | CTA 直下 |
| 30秒で | 強 | CTA 直下 |
| いつでも解約 | 中 | 料金表・FAQ |
| 安心・安全 | 中 | セキュリティ説明 |
| パスワード不要 | 中 | OAuth 説明 |

### TrendBird 実例

- 「クレジットカード不要・30秒で開始」— LP CTA 直下
- 「OAuth 2.0 PKCEフローを採用しており、パスワードを当サービスに共有することはありません」— FAQ

### 使い方のコツ

- CTA の直下に安心コピーを配置するのが最も効果的
- 具体数字（30秒、3ステップ）で「かんたんさ」を裏付ける
- セキュリティ系は技術用語を避け、ユーザーメリットで説明する

---

## 3. 低コミット

「試すだけ」という心理的ハードルの低さを伝える言葉。

### 基本ワード

| ワード | 用途 |
|--------|------|
| まずは | CTA・導入文 |
| 試してみる | CTA |
| 始めてみる | CTA |
| いつでもやめられる | 安心コピー |
| 登録するだけ | ステップ説明 |
| ワンクリックで | 機能説明 |

### TrendBird 実例

- 「無料で始める」— 最も低コミットな CTA
- 「ワンクリックで投稿」— LP How it Works
- 「興味のあるジャンルを選び、監視したいトピックを登録するだけ」— LP ステップ1

### 安心感との組み合わせ

低コミット + 安心感の掛け合わせが最強パターン:
- 「無料で始める」（低コミット）+「クレジットカード不要」（安心感）
- 「まずは試してみる」（低コミット）+「いつでも解約」（安心感）

## 4. 社会的証明

「他の人もやっている」という安心材料を提供する言葉。

### 基本ワード

| ワード | 用途 | 注意 |
|--------|------|------|
| 話題の | LP・X投稿 | トレンド文脈で自然に使える |
| 注目の | LP・X投稿 | |
| 〜人が利用 | LP | 実数がある場合のみ |
| 選ばれている | LP | ユーザー数が少ない初期は避ける |
| おすすめ | 料金表 | バッジとして使用 |

### TrendBird 実例

- 料金表の「おすすめ」バッジ — 社会的証明の簡易版

### 初期フェーズの注意

- ユーザー数が少ない段階で「〜人が利用」は使わない
- 代わりに「Xで話題のトレンドを」のように対象の社会的証明を借りる
- 開発者の実体験（「自分で使って便利だった」）も社会的証明の一種

## 5. 共感

「わかる」「自分もそう」という感情を引き出す言葉。

### 基本ワード

| ワード | 用途 |
|--------|------|
| 〜って経験ないですか | X投稿フック |
| あるある | X投稿フック |
| 〜ですよね | X投稿・LP |
| 実は〜 | X投稿フック |
| 正直に言うと | X投稿（本音系） |

### 感情ワード（X投稿向け）

**ポジティブ系:**
- すごい、面白い、最高、嬉しい、楽しい、ワクワク、感動

**柔らかい断定:**
- 〜だと思う、〜な気がする、〜かもしれない

**余韻系:**
- …（三点リーダ）、〜なんだよな、〜って話

### TrendBird 実例

- 「トレンド追いきれなくなってきた…って経験ないですか」— 課題共感
- チュートリアル全体のカジュアルトーン — 親しみの共感

### 日本市場での効果

- **共感は日本のXで最もエンゲージメントを生む感情トリガー**
- 「いいね」の大半は共感から来る
- 断定より問いかけ（「〜ですよね？」）が効果的

## 6. 具体数字

抽象的な表現を具体的にし、説得力を高める。

### 基本パターン

| パターン | 例 | 効果 |
|---------|-----|------|
| N ステップ | 3ステップ | 手順の簡潔さ |
| N 秒/分 | 30秒で開始 | 時間の短さ |
| N 回/月 | 1日1回の通知 | 頻度の具体化 |
| N 倍 | リプライの重み150倍 | インパクト |
| N 件/個 | 最大10個まで | 制限の明確化 |

### TrendBird 実例

- 「かんたん3ステップ」— LP How it Works
- 「30秒で開始」— CTA 直下
- 「1日1回の通知」— 通知説明

### 使い方のルール

1. **丸め数字は避ける** — 「約100件」より「98件」の方が信憑性がある
2. **比較対象を示す** — 「150倍」だけでなく「いいねの150倍」
3. **UI では制限を明確にする** — 「少ない」ではなく「最大3個まで」
4. **LP では効果を数字化する** — 「すぐ始められる」より「30秒で始められる」
